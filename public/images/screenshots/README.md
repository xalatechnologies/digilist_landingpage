# Application Screenshots

Place your application screenshots in this directory:

- `desktop-app.png` - Desktop application screenshot (recommended: 800x600px or larger)
- `tablet-app.png` - Tablet application screenshot (recommended: 600x800px or larger)
- `mobile-app.png` - Mobile application screenshot (recommended: 375x667px or larger)

The images will be automatically displayed inside device mockups in the hero section.
